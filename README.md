# 🎬 CineVault

CineVault is a Java-based desktop application that allows users to manage and organize their personal movie collections. Each user has their own account and can maintain a separate list of movies, ensuring privacy and personalized tracking.

---

## 🚀 Features

- 🔐 User Authentication (Login/Register)
- 🎥 Add, Update, Delete Movies
- ⭐ Rate Movies
- 📂 View Movies by Rating
- 📊 Track Total Movies & Genre-wise Count
- 🗄️ SQLite Database Integration

---

## 🛠️ Tech Stack

- **Language:** Java
- **UI:** (Swing / Console – based on your implementation)
- **Database:** SQLite
- **Architecture:** MVC (Model-View-Controller)
- **Pattern Used:** DAO (Data Access Object)
- **JDBC:** For database connectivity
